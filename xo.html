Noughts And Crosses Game



Project Overview

For my second year Further Programming and Problem Solving module I was tasked with creating a Noughts and Crosses game for Android Devices. I used Android Studio and the Java language to develop the game.

The game had to have a algorithm that made an attempt to block the players move and attempt to stop them from winning. It also had to have the functionality for the user to choose the computer to make the first move, and had to show game statistics, like the number of games played, and win percentage.

It's imprtant to note that at the time of making this game, I was still a novice to programming concepts and programming in general, but this piece of work demonstrates experience with the Java language.

Skills Developed
  • Development of Java Programming Language
  • Familiarity of Anroid Studio IDE
  • Understanding of writing functional code
  • Making simple animations from images
  • Linking up game logic with a user interface
