Tennis App



Project Overview

For my third year Advanced Programming module I was tasked with creating an iOS application using Swift 4 that kept track of and calculated scores, for a men's singles tennis match.

My overall grade from the moudles assignments (assingments made up 63% of the overall module mark) was 82%.


A starter project was provided, with a UI already built, and no changes were to be made to the UI. The features I implemented were as followed:

  • Allow the user to indicate which player scored a point, and correctly display the correct score for the current game, set and match, along with showing all preceeding set scores
  • Detect the end of the game, and progress automatically
  • Detect the end of the set, and progress automatically
  • Detect the end of the match and prevent further interaction until the user restarted the match
  • Notify the user of the winner of the match, and have a completed score board for the match
  • Handle Tie Breaks
  • Handle the requirement for a player to win by 2 clear games in the fifth set
  • Allow the user to reset the match at any point
  • Use text to speech to call new balls please after the first 7 games and then after every 9 games
  • Identify who should serve next at any given point, by changing the background colour of the player name label
  • Audible alert to indicate the change in server
  • Include comprehensive unit tests
Additionally it was a requirement to use good object-oriented principles and adhere to the DRY principle

  • Further enhancement of the Swift language
  • Understanding of OOP principles
  • Understanding of SOLID principles
  • Adhering to the DRY principle
